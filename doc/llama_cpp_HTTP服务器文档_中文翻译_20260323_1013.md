# LLaMA.cpp HTTP 服务器

基于 httplib、nlohmann::json 和 llama.cpp 的快速、轻量级、纯 C/C++ HTTP 服务器。

提供一套 LLM REST API 和 Web UI 用于与 llama.cpp 交互。

## 功能特性

- 支持 F16 和量化模型在 GPU 和 CPU 上的 LLM 推理
- 兼容 OpenAI API 的聊天补全、响应和嵌入路由
- 兼容 Anthropic Messages API 的聊天补全
- 重排序端点 (#9510)
- 支持多用户的并行解码
- 连续批处理
- 多模态（文档）/ 兼容 OpenAI API
- 监控端点
- 模式约束的 JSON 响应格式
- 类似 Claude API 的助手消息预填充
- 支持几乎所有模型的函数调用 / 工具使用
- 推测解码
- 易于使用的 Web UI

完整功能列表请参阅服务器的更新日志
